――少し、息苦しかった。
ゆっくりと、目を開いた。
ひどく寒い。
吐き出した息が白く曇る。
遠くからは、断続的に工事をしているような音が聞こえていた。
視界には、満天の星空。
こんなにくっきりとした星空は、今まで見たことがない。
三日月も、星々に圧倒されたかのように、申し訳無さそうな細さで浮かぶばかりだ。
<span class="name">七海</span>:「おにぃ……目が覚めた？」
七海が、僕を見下ろしていた。
その顔は、少し疲れているように見える。
髪や、着ている服は、埃で汚れてしまっている。
そんな七海の目からこぼれ落ちた一滴の涙が、僕の頬を濡らした。
<span class="name">七海</span>:「よかった……」
どうやら僕は、七海に膝枕をしてもらっていたようだ。
<span class="name">拓巳</span>:「……僕は、帰ってきたの？」
確認するようにつぶやくと、七海は泣き笑いの表情で、小さくうなずいた。
改めて、周囲を見回す。
僕が横たわっていたのは、瓦礫の山の中。
遠くには、崩壊したビル群があるはずだけど、今は夜の闇のせいでよく見えない。
街から明かりが消えたせいで、あんなにもはっきりと星が見えたのかもしれない。
少し離れたところで斜めに傾いているコンテナが、僕が住んでいた<span class="dict">ベース</span>だと気付いた。
そうか、ここは、ＫＵＲＥＮＡＩビルの屋上なんだ。
なぜ僕と七海はここにいるのか。
梨深たちはどこにいるのか。
そんな疑問も浮かんだけど、すぐにどうでもよくなった。
七海がそばにいてくれた。
こうして現実の渋谷で、また七海と無事に再会できた。
それが、嬉しい。
<span class="name">拓巳</span>:「覚え……てるか？　あの、夏の、渋谷のこと」
<span class="name">七海</span>:「うん……」
<span class="name">七海</span>:「おにぃと、ラブラブになれたよね」
<span class="name">七海</span>:「忘れるわけ、ないよ」
<span class="name">七海</span>:「あんなに幸せだった時間を、忘れたくなんてない……」
七海の手が、僕の頬を撫でる。
その“右手”は、冬の寒さのせいでとても冷たくて。
僕は自分の手を、七海の手に重ねた。
<span class="name">拓巳</span>:「これからだって、幸せでいられるよ……」
<span class="name">拓巳</span>:「僕は、七海と星来がそばにいてくれたら、それだけで、幸せなんだ……」
<span class="name">七海</span>:「ナナも……おにぃとずっと一緒にいたい」
<span class="name">七海</span>:「もう、置いていかないでね？」
<span class="name">拓巳</span>:「うん……。絶対に、離さない……」
<span class="name">七海</span>:「約束だよ」
儚げな笑みを浮かべて。
七海が、僕に覆い被さるようにして、キスをしてくれた。
<span class="name">七海</span>:「バカおにぃ……大好き」

あれから２日。
僕はずっと家に引きこもっていた。学校にも、ブルゥムーンにも行ってない。
あんな事件があって、どんな顔でりょーたすたちに会えばいいのか、わからなかった。
かと言って、何もしないのも落ち着かなくて、僕はただ淡々とサライの見つけた解読法で、ＣＯＤＥを読み解いていた。
<span class="name">ゾン子</span> : アンタさあ、いつまでそんなことやってるつもり？
いきなりゾン子が話しかけてきた。
<span class="name">悠太</span> : ……うるさいな
<span class="name">ゾン子</span> : 他にやることあるんじゃないの？　あの子のことほっといていいの？
ゾン子の声が耳に、胸に刺さる。でも僕はそれを振り払って、ひたすら解読に集中した。
だって、今の僕がみゅうポムに会いに行って、何ができるって言うんだ。
だったら、なんでもいいから自分にできることをする方がマシじゃないか。
<span class="name">ゾン子</span> : そんなの他の誰かに任せた方がよくない？
<span class="name">悠太</span> : 今のところ４３人分の名前が一致してるんだ。最終的には全員が一致するはずだ
<span class="name">ゾン子</span> : そんなリストよりも鍵よ、鍵。そっちの方が大事でしょ？
ゾン子の怒りを表すみたいに、一際、強くノイズが走った。なんなんだよ、ゾン子のヤツ、やけに機嫌が悪いな。
<span class="name">悠太</span> : そりゃ鍵だって大事だけど、散々あちこち歩き回って探したって、鍵穴なんてどこにも見つからなかったんだからさ
<span class="name">悠太</span> : 今やるべきことはこれだ……これしかないんだ
あの『コトリバコ』に鍵穴はなかった。
家の近所にある鍵穴はだいたい試しちゃったし、こうなると打つ手がない。
僕はゾン子を無視してひたすら解読作業に没頭した。
<span class="name">悠太</span> : ……『ニシゾノリリカ』っと。……ってニシゾノリリカ！？
しばらくして、５５人目の名前を解読した瞬間。僕はびっくりして目を見開いた。
ニシゾノリリカって、西園梨々花？　いや、同一人物だと限ったわけじゃないし……。
急に沸いてきた不安をぐっと抑え、解読を進めていく。でも、その不安は新しい名前が１つ増えるごとに大きくなっていった。
『ニシゾノリリカ』『ハシガミサライ』『アイカワミユウ』『ミナセリア』『スミカゼトウコ』『モリツカシュン』……。
なんだよ、これ。なんで、僕の知ってる名前ばっかり出てくるんだよ。入ってないのは、僕とりょーたすくらいだ。
<span class="name">悠太</span> : って、まさか僕の名前まで入ってたりしないよね……ははっ
だけど、新しく解読した文字を見たら。
『ＧＡＭＯＮ』って書いてあって。
<span class="name">悠太</span> : ガモン……？
続けて、『ＹＵＴＡ』って書いてあった。
<span class="name">悠太</span> : ユータ……？
嘘、まさか、本当に僕の名前まであった……！？

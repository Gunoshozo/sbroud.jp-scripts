”南方　ひづる”
”8月20日”
<span class="name">南方ひづる</span>: 今日で日都ヶ島ともお別れやな。
<span class="name">南方ひづる</span>: （あれからの日々は、　長かったような短かったような……）
<span class="name">南方ひづる</span>: （少なくとも私の生活が　一変してしもたことはたしかや）
<span class="name">南方ひづる</span>: （あれ以来、藁の声は聞こえない。　未来に帰ったんやろか？）
<span class="name">男の声</span>: 行くんやな。
<span class="name">南方ひづる</span>: 根津……。
<span class="name">根津銀次郎</span>: 呼び捨ては止めんかい。なんぼ言うてもわからんやっちゃな。
<span class="name">南方ひづる</span>: 気にせんといて。
<span class="name">根津銀次郎</span>: 気にするわ！
<span class="name">根津銀次郎</span>: ふん……まあええわい。行き先はどこや。
<span class="name">南方ひづる</span>: 東京へ引っ越すことになったわ。父が東京にあてがあるらしい。
<span class="name">根津銀次郎</span>: 東京か。遠いのォ。
<span class="name">南方ひづる</span>: 両親が言うんよ……。この島にいるのはもう辛いてな。
<span class="name">根津銀次郎</span>: そらしかたないやしょ。そう言うおまえはどうなんじゃ。
<span class="name">南方ひづる</span>: …………。
<span class="name">根津銀次郎</span>: おまえも、そうやないんか？
<span class="name">南方ひづる</span>: 私自身にそんな感情はあらへん。どっちか言うたら……。
<span class="name">南方ひづる</span>: 竜之介をこの島におらせたない、そんなとこや。
<span class="name">根津銀次郎</span>: …………。
<span class="name">南方ひづる</span>: まだ、あの話が信じられへんみたいやな。
<span class="name">根津銀次郎</span>: しゃあないやろ。あまりにも突飛な話やさけよォ。
<span class="name">南方ひづる</span>: そうやな。亡くなった故人を「心の中にいる」と表現することは多々あるけど……。
<span class="name">南方ひづる</span>: 比喩表現でもなんでもなく、実際に亡くなった人間が体の中におるやなんて、にわかに信じられへんやろな。
<span class="name">根津銀次郎</span>: けどこの島には、あり得へんことがあるっちゅうんは、実際に見てよう知っとる。
<span class="name">南方ひづる</span>: そうか……。
<span class="name">根津銀次郎</span>: このまま、あの化物を野放しにはできん。儂はこの島で、あの化物を狩り続けるで。
<span class="name">南方ひづる</span>: 化物相手でも、猟師らしい思考やな。
<span class="name">根津銀次郎</span>: 倒し方はもうわかっちゃあるさけな。あんなもんにのさばられてたまるかい。
<span class="name">根津銀次郎</span>: そのうちヤツらの習性や有効な攻撃、それに武器なんかもわかってくるはずや。
<span class="name">根津銀次郎</span>: 早々にくたばらんかったら、の話やがな。
<span class="name">南方ひづる</span>: そうか……。でもこの先、頻繁にヤツらと遭遇することはあらへんと思う。
<span class="name">根津銀次郎</span>: そら、なんでや？
<span class="name">南方ひづる</span>: 連中はずいぶん昔からこの島にいる可能性が高い。
<span class="name">南方ひづる</span>: それなのに広く知られずにいたということは、これまでずっと目立たないよう、派手な活動を控えてきたんやろな。
<span class="name">南方ひづる</span>: せやから、誰も彼も無差別に襲うなんてことは、めったにしやんのやろ。
<span class="name">南方ひづる</span>: この島は狭い。人が一人行方不明になっただけで大事や。すぐに事件として周知されてまう。
<span class="name">南方ひづる</span>: だからこそ化物は大っぴらに活動してこんかった。それはあのハイネかて同じや。
<span class="name">南方ひづる</span>: だからこそ、人を避けてた。だからこそ、人に飢えてたんや。
<span class="name">根津銀次郎</span>: なるほどな……。
<span class="name">南方ひづる</span>: 根津が相手する化物は、飢えた肉食獣より厄介な連中や。気ィつけてな。
<span class="name">根津銀次郎</span>: 心配せんでも、おまえのような無茶はせん。
<span class="name">南方ひづる</span>: 暇ができたら手紙出すわ。メールや電話より手紙のがええやろ。
<span class="name">南方ひづる</span>: そっちも気が向いたら、手紙で猟の成果でも教えてや。
<span class="name">根津銀次郎</span>: 知ってどないすんや？
<span class="name">南方ひづる</span>: あれだけのことがあった相手や。情報だけでも集めときたいて思ても不思議やないやろ。
<span class="name">根津銀次郎</span>: ふん……。まあ、気が向いたらな。
<span class="name">南方ひづる</span>: ああ、助かる。あの日のこと、ハイネのこと……まだまだわからんことが多すぎるからな。
<span class="name">南方ひづる</span>: そろそろ行かな。両親を待たせてるんよ。
<span class="name">根津銀次郎</span>: もう、戻ってくるんやないぞ。
<span class="name">南方ひづる</span>: ああ、この物語が、そうさせてくれるならな。
<span class="name">根津銀次郎</span>: またワケのわからんこと言いよって。
<span class="name">南方ひづる</span>: それじゃあ、お別れや。
<span class="name">南方ひづる</span>: あの日、助けてくれてありがとう。きっとこの先も、私はこの出会いに感謝し続けると思う。
<span class="name">根津銀次郎</span>: ……ふん、おまえが言うと、なんぞ裏がありそうやで。
<span class="name">南方ひづる</span>: 失礼やな。まあ、そのくらいに受け取ってもろたほうが、こっちも気ィ楽やけどな。
<span class="name">南方ひづる</span>: それじゃ、今度こそほんまのサヨナラや。
<span class="name">根津銀次郎</span>: ああ、達者でな。
<span class="name">南方ひづる</span>: （見送りに来てくれた朝子が、　泣きながらずっと手を振ってくれてる……）
<span class="name">南方ひづる</span>: （根津は……もうおらんか）
<span class="name">南方ひづる</span>: 日都ヶ島……だんだん小さなっていきよる。
<span class="name">南方ひづる</span>: （ここから先は自分自身で　未来を切り拓いていくんや）
<span class="name">南方ひづる</span>: （そうして歩んでいく道のりが、　いつかまた……）
<span class="name">南方ひづる</span>: （あの島へと繋がってくんやろな）
<span class="name">南方ひづる</span>: （そう……。　望むと望まざるにかかわらず）
<span class="name">南方ひづる</span>: さよならハイネ。次会うた時は……
<span class="name">南方ひづる</span>: 正々堂々勝負できたらええな。

There are 30 hidden radio tranmissions in a game. You can pull up sky sensor when you see this symbol at the end of the text box:

I believe, the game creates a quick save when those appear.

You need to get all tranmissions to reach a sertain ending

To branch to the various routes you will need to memorise(ingame mechanic) sertain words. Those are highlighted in yellow in the game, and also highlighted on this site.


Paradigm Change Bad END

Ignore highlighted words and don't write any articles


Arcana Dream END (Miyuu Route)

Memorize highlighted words: ひとりかくれんぼ → コックリさん → ミイラ

Write first article: コックリさん → image with circle (3rd one)

Memorize highlighted words: 口コミ → 美人過ぎる → 占い

Write second article: 占い → 口コミ → 美人過ぎる → image with tiles lied in a circle (4th one)


Summoner Soul END (Aria Route)

Memorize highlighted words: ひとりかくれんぼ → コックリさん → ミイラ

Write first article: ミイラ → image with circle (3rd one)

Memorize highlighted words: 口コミ → 黒魔術、代行屋 → 釣り → 美人過ぎる → カード → メール → 超常現象 (replace コックリさん) → 運勢 (replace ひとりかくれんぼ) → マスメディア (replace ミイラ)

Write second article: 黒魔術代行屋 → 口コミ → 美人過ぎる → only image with 4 cartoon characters

Write third article: 黒魔術代行屋 → マスメディア → 美人過ぎる → only image with 4 cartoon characters

Memorize highlighted words: 偽物 (replace 黒魔術代行屋) → 民間伝承 (replace カード)

Write forth article: 民間伝承 → 口コミ → 超常現象 → only image "rectangles with random patterns"


Mumu Smile END (Touko Route)

Memorize highlighted words: ひとりかくれんぼ → コックリさん → ミイラ

Write first article: ひとりかくれんぼ → image with eye (1st one)

Memorize highlighted words: 口コミ → → 釣り → 美人過ぎる → カード → メール → ムムー → 超常現象 (replace コックリさん) → 運勢 (replace ひとりかくれんぼ) → マスメディア (replace ミイラ)

Write second article: ムムー → 釣り → 超常現象 → only image "A couple of bold lines with the middle forming a triangle"

Write third article: ムムー → マスメディア → 超常現象 → only image "A couple of bold lines with the middle forming a triangle"

Write forth article: ムムー → 口コミ → 超常現象 → only image "A couple of bold lines with the middle forming a triangle"

Memorize highlighted words: ほどよい (replace ムムー) (Why? IDK)
